Machine Learning based upon decision trees

Part of my final project is building a machine learning program based upon decision trees. This was done using C++ and using data sets obtained from the UCI machine learning repository. This program works well with binary sets, and explodes when considering complex data sets. It is currently being refactored to handle more complex data sets, and to be data agnostic.

Fuzzy Logic controller

I created a fuzzy logic control system to solve a virtual pendulum and cart problem. I really enjoyed this project, although by the time I submitted it, I was only 'mostly' successful. The pendulum balances, but then slowly wanders off to one side with infinite happiness. I'm interested in readdressing this problem when time permits.

8-puzzle problem

The 8-puzzle problem was used as a method of exploring different space-state search algorithms. Despite the simplicity of this project, I really enjoyed it. It greatly enhanced my knowledge of the algorithms used. These included breadth-first search, breadth-first search with visited list, progressive deepening search, uniform cost search with expanded list and A* search with expanded list.
